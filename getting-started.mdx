## Installation and Setup

You start your Paperguide journey by creating an account and initializing your documentation space. This process takes less than five minutes and requires only an email address. Once set up, you customize your workspace to fit your project's needs.

<Steps>
  <Step title="Create Account" icon="user">
    Visit the Paperguide website and sign up with your email. Verify your account to access the dashboard.
    
````bash
    # No installation needed - web-based access
    # Simply navigate to app.paperguide.com
    `````
  </Step>
  <Step title="Initialize Workspace" icon="settings">
    Name your space and add initial folders for organization.
    
    <Callout kind="tip">
      Choose a descriptive name like `ProjectAlpha-Docs` for easy identification.
    </Callout>
  </Step>
  <Step title="Invite Collaborators" icon="users">
    Share access with your team via email invites. Set roles such as editor or viewer.
  </Step>
</Steps>

## Creating Your First Document

You create documents using the built-in editor, which supports rich text and Markdown. Add sections, images, and code blocks to build comprehensive guides.

<Tabs>
  <Tab title="Markdown Mode" icon="code">
    Switch to Markdown for advanced formatting.
    
````markdown
    # My First Doc
    
    ## Section One
    This is a sample section with `inline code`.
    
    ```javascript
    // Nested code example
    function initDoc() {
      console.log('Document ready');
    }
    initDoc();
    ```
    `````
  </Tab>
  <Tab title="Visual Editor" icon="edit">

## Troubleshooting Common Issues

If you encounter login problems, clear your browser cache. For collaboration sync issues, check your internet connection.
