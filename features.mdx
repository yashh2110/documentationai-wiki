## Collaboration in Action

You invite users and assign permissions to control access. Changes appear instantly, with notifications for updates.

<Callout kind="success">
  Real-time cursors show who is editing what section.
</Callout>

<Steps>
  <Step title="Invite Users" icon="mail">

````javascript
    // Simulated invite API call

    });
    `````

  </Step>
  <Step title="Set Permissions" icon="lock">
